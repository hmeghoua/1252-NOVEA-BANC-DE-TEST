\doxysubsection{Detailed Description}
X\+ML text. A text node can have 2 ways to output the next. \char`\"{}normal\char`\"{} output and C\+D\+A\+TA. It will default to the mode it was parsed from the X\+ML file and you generally want to leave it alone, but you can change the output mode with \mbox{\hyperlink{class_ti_xml_text_acb17ff7c5d09b2c839393445a3de5ea9}{Set\+C\+D\+A\+T\+A()}} and query it with \mbox{\hyperlink{class_ti_xml_text_aac1f4764d220ed6bf809b16dfcb6b45a}{C\+D\+A\+T\+A()}}.
