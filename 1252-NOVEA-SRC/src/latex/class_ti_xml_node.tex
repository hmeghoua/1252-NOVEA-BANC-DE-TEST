\doxysubsection{Detailed Description}
The parent class for everything in the Document Object Model. (Except for attributes). Nodes have siblings, a parent, and children. A node can be in a document, or stand on its own. The type of a \mbox{\hyperlink{class_ti_xml_node}{Ti\+Xml\+Node}} can be queried, and it can be cast to its more defined type.
