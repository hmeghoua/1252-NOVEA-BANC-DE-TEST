\doxysubsection{Detailed Description}
Print to memory functionality. The \mbox{\hyperlink{class_ti_xml_printer}{Ti\+Xml\+Printer}} is useful when you need to\+:


\begin{DoxyEnumerate}
\item Print to memory (especially in non-\/\+S\+TL mode)
\item Control formatting (line endings, etc.)
\end{DoxyEnumerate}

When constructed, the \mbox{\hyperlink{class_ti_xml_printer}{Ti\+Xml\+Printer}} is in its default \char`\"{}pretty printing\char`\"{} mode. Before calling Accept() you can call methods to control the printing of the X\+ML document. After \mbox{\hyperlink{class_ti_xml_node_acc0f88b7462c6cb73809d410a4f5bb86}{Ti\+Xml\+Node\+::\+Accept()}} is called, the printed document can be accessed via the \mbox{\hyperlink{class_ti_xml_printer_a859eede9597d3e0355b77757be48735e}{C\+Str()}}, Str(), and \mbox{\hyperlink{class_ti_xml_printer_ad01375ae9199bd2f48252eaddce3039d}{Size()}} methods.

\mbox{\hyperlink{class_ti_xml_printer}{Ti\+Xml\+Printer}} uses the Visitor A\+PI. \begin{DoxyVerb}TiXmlPrinter printer;
printer.SetIndent( "\t" );

doc.Accept( &printer );
fprintf( stdout, "%s", printer.CStr() );
\end{DoxyVerb}
